# RakshaPeeth

Protecting people, simplifying security.

RakshaPeeth is an automated gate security system designed for residential universities. It enables secure entry and exit management using AI-based verification, real-time tracking, and instant notifications.

## Features
- AI-based verification (Iris / Number Plate Recognition)
- Real-time entry/exit tracking
- Parent notifications (WhatsApp alerts)
- Role-based access (Admin, Guard, Parent)
- Smart visitor management system

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- AI Services: Flask, FastAPI, YOLOv8
